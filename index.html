Overview of Jenkins X

Jenkins X is a CI / CD platform for Kubernetes which includes:

Helm Chart

Jenkins X is available as a cloud native helm chart for easy installation onto any kubernetes cluster

Pipelines

Reusable pipeline flows that work out of the box for most common languages for full CI / CD on kubernetes

Environments

Migrates your application through separate environments via helm and GitOps

CLI

A CLI tool to help you quickly create new microservices or setup CI / CD with Jenkins X on your existing repositories

Apps

A collection of software tools packaged as helm charts that makes it super easy to use Jenkins X with tools like Nexus, Artifactory, SonarQube, Prometheus, Elasticsearch, Grafana etc
